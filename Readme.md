## Supported:

### Database

- Postgres
- SQLite
- MySQL
- Clickhouse
- MongoDB
- Aerospike

### Cache

- Memory
- Redis
- Memcached

### Servers

- http
- Kafka

### Server middlewares

- Restore from panic
- Register timing middleware (default use)

### Clients

- http
- Kafka

### Other

- Initialize base engine module from yaml config file
